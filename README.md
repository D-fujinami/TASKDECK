# TASKDECK-custom

TASKDECK-custom は、夢眠 ねるこ( ⩌⩊⩌ ) 様が制作・公開されている **TASKDECK** をベースに、個人利用を中心に機能追加・改善を行っている Fork です。

オリジナルの「軽量・オフライン・個人向け」というコンセプトを尊重しつつ、IT コンサルタント、PMO、SRE、個人事業主など、複数顧客・複数案件を並行して管理する用途に合わせた改善を進めています。

## 改修方針

基本方針は以下です。

- オフラインで利用できること
- シンプルな操作性を維持すること
- 重量級のプロジェクト管理ツールにはしないこと
- オリジナルのUI/UXを尊重すること
- 小さな改善を積み重ねること

---
